# Περιγραφή Παιχνιδιού

Το παιχνίδι στρατηγικής Σκορ 4 ή αλλιώς Connect four αποτελείται από ένα ταμπλό διαστάσεων 7×6 (στήλες × γραμμές) και παίζεται από 2 παίκτες.
Κάθε παίχτης έχει στην διάθεσή του 21 μάρκες, οι οποίες τοποθετούνται σε οποιαδήποτε στήλη από την κορυφή του ταμπλό και ολισθαίνουν προς τα κάτω. Σου επιτρέπει να αναπτύξεις τις στρατηγικές σου ικανότητες και να κερδίσεις τον αντίπαλο, σχηματίζοντας μια σειρά από 4 μάρκες ίδιου χρώματος προς οποιαδήποτε κατεύθυνση (οριζόντια, κάθετα και διαγώνια). Σκέψου καλά ποια θα έιναι η επόμενη κίνηση! Δεν πρέπει μόνο να φτιάχνεις τετράδες αλλά να κόβεις και τις τετράδες που πάει να δημιουργήσει ο αντίπαλος!
